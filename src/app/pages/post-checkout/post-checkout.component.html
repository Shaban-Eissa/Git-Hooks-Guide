Post-Checkout Hook Setup

Triggered after checking out a branch. Great for syncing environment or regenerating files automatically.

🔗

Hook Setup

Create the hook file: .husky/post-checkout

bash scripts/post-checkout.sh "$1" "$2" "$3"
